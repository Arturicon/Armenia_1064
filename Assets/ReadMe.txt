1. ExtraInjectAttribute, IExtraInject - данная связка существует только для объектов обитающих в глобальном контексте. Надо для того, чтобы на каждой новой сцене
внедрять в глобальный объект экземпляр объекта который на каждой сцене свой. Т.е при переходе на новую сцену класс ExtraInjector внедряет в глобальный класс те зависимости
который у того помечены атрибутом ExtraInject.
IExtraInject - объекты которые реализуют его должны быть привязаны к контексту через него как AsSingle или AsCached. 
Пока не работает на объекты создаваемые через zenject фабрики

2. IGameElement - реализуется для того чтобы реагировать на стадии игры в целом и сцены в частности (начало, конец, пауза и тд). 
Интерфесы начинающиеся с IScene (например ISceneFinish) нельзя реализовывать у классов существующих в глобальном контексте (MonoGlobalInstaller)
Интерфесы начинающиеся с IGame (например IGameLeave) могут реализовывать любые классы

3. ConfigurationRuntime - Это класс посредник нужен для того, чтобы можно было менять некоторые параметры в процессе игре, и чтобы они
не менялись при этом перманентно в ScriptableObject настройках
